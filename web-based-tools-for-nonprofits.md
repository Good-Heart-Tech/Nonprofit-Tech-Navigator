# Web-Based Tools for Nonprofits

[Good Heart Tech](https://goodhearttech.org/) offers free web-based tools for nonprofits. These tools help with passwords, files, websites, and public record lookups.

### 🌟 Start here

* [**Nonprofit Tools Hub**](https://nonprofittools.org/): A simple place to explore all of these tools.
* [**Nonprofit Monitor**](https://nonprofitmonitor.org/): Look up nonprofit details and common technologies using IRS and other public data.

### 🔐 Security tools

* [**Password Pusher**](https://push.goodheart.tech/): Share passwords with encrypted links that expire automatically.
* [**Password Generator**](http://password.nonprofittools.org/): Create strong, easy-to-type passphrases.

### 📄 File and media tools

* [**QR Code Generator**](https://qr.goodheart.tech/): Create QR codes for websites and Wi-Fi networks. [See on GitHub](https://github.com/Good-Heart-Technology/QR-Code-Generator)
* [**Stirling PDF**](https://pdf.goodheart.tech/): Merge, split, compress, and convert PDF files.
* [**File Converter**](https://convert.nonprofittools.org/): Convert documents, images, and media files between common formats.
* [**Wallpaper Generator**](https://wallpaper.nonprofittools.org/): Create simple branded wallpapers with your logo and colors.

### 📊 Performance and analysis

* [**Domain Analyzer**](https://dns.nonprofittools.org/): Review public DNS and email-related records for security and deliverability checks.
* [**Speed Test**](https://speed.goodheart.tech/): Measure your internet connection speed.
* [**Web Check**](https://webcheck.goodheart.tech/): Inspect website security, performance, and technical details.
