# Managing Mail in Outlook (Microsoft 365)

Email is one of the most critical tools for nonprofit communication. Outlook, included with Microsoft 365, offers powerful features to help you stay safe, stay organized, and stay on top of your work. This guide focuses on how to manage mail effectively and securely.

### 🔐 Use the "Report" and "Block" Options to Stay Safe

Outlook includes built-in tools to help identify and block malicious or unwanted messages. You can access these options in two places:

* **Home tab** → _Report_ button
  *

      <figure><img src="../../.gitbook/assets/image (54).png" alt="" width="141"><figcaption></figcaption></figure>
* **Right-click on any email** → _Report_ or _Block_ submenus menu
  *

      <figure><img src="../../.gitbook/assets/image (55).png" alt=""><figcaption></figcaption></figure>

#### Available actions:

* **Report Phishing** – flags deceptive messages to Microsoft.
* **Report Junk** – categorizes spam or unwanted email.
* **Block Sender** – prevents future emails from a specific sender.
* **Never Block Sender** – ensures important senders aren’t mistakenly filtered.

#### Why it matters:

Using these tools helps improve Microsoft’s filters and protects others in your organization from similar threats. Blocking known spam and whitelisting key senders improves inbox reliability for everyone.

### 📁 Organize Your Inbox with Folders and Subfolders

Folders help you stay on top of your work and reduce clutter. Organize emails based on teams, projects, or topics.

#### Best Practices:

* Create high-level folders like `Donors`, `Volunteers`, `Grants`, `Operations`.
* Use subfolders for finer sorting: `Grants > Submissions`.
* Avoid too many layers—simple structures are easier to manage.

> Pair folders with **Rules** to automatically sort messages into the right location.

### 🗂️ Create Rules to Automate Your Inbox

Rules automatically sort, flag, or take action on incoming messages.

#### To create a rule:

1. Right-click a message.
2. Select **Rules > Create Rule...**
3. Choose your condition (e.g., from a sender or specific words).
4. Choose the action (e.g., move to a folder).
5. Save the rule.

#### Common use cases:

* Move donation confirmations to a `Fundraising` folder.
* Flag emails with "Time Sensitive" in the subject.
* Route newsletters to a "Read Later" folder.

### 📫 Shared Mailboxes: A Smart Tool for Teams

Shared mailboxes (like `info@yourorg.org` or `volunteers@yourorg.org`) allow multiple team members to send and receive emails from a single address. They're perfect for:

* Coordinating responses to external inquiries
* Maintaining continuity when staff change roles
* Keeping communication centralized for departments or roles

If one has already been set up for you, here’s how to access it: 🔗 [Accessing a Shared Mailbox in Microsoft Outlook](https://learn.goodhearttech.org/microsoft-365/microsoft-365-user-guides/accessing-a-shared-mailbox-in-microsoft-outlook)

{% hint style="warning" %}
**Ask your IT administrator to create a shared mailbox** if your team needs one.  It's recommended that all Shared Mailboxes have their login DISABLED for security purposes.  Delegates can still access them using their named accounts.
{% endhint %}
