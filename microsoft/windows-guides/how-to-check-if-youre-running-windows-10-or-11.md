# How to Check if You’re Running Windows 10 or 11

### Option 1: Check in Settings

1. Click the **Start Menu** (Windows icon in the bottom-left corner).
2. Select **Settings** (gear icon).
3. Go to **System** → **About**.
4. Under **Windows specifications**, look for **Edition** and **Version**.
   * If it says **Windows 10**, your computer is running Windows 10.
   * If it says **Windows 11**, your computer is running Windows 11.

### Option 2: Use a Quick Command

1. Press **Windows Key + R** on your keyboard.
2. Type `winver` and press **Enter**.
3. A small window will appear showing your Windows version and edition.
