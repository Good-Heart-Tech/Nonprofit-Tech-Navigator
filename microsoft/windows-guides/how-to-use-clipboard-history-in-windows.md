# How to Use Clipboard History in Windows

Clipboard history saves multiple things you copy, not just the last one.

### Turn on Clipboard history

1. Press **Win + V**.
2. Click **Turn on**.

That’s it. You only need to do this once per PC.

{% hint style="info" %}
If you don’t see a “Turn on” button, it’s already enabled.
{% endhint %}

### Use Clipboard history

1. Copy text or images like normal.  `Ctrl + C`, right-click **Copy**, etc.
2. Press **Win + V**.
3. Click an item to paste it.

#### Power tips

* Use your arrow keys to select an item.
* Press **Enter** to paste the selected item.
* Click the **pin** icon to keep an item.  Pinned items stay, even after reboots.

### Clear your clipboard history (optional)

1. Press **Win + V**.
2. Select **Clear all**.

{% hint style="warning" %}
Clearing the list usually won’t remove pinned items. Unpin first if you want them gone.
{% endhint %}
