# How to Enable Cookies for a Website

Some websites need cookies to sign you in, remember your session, or complete security checks. This can help fix pages that will not load correctly, keep signing you out, or fail a security check.

{% hint style="info" %}
This example uses Google Chrome. Other browsers usually follow a similar process, but the menu names and wording may be a little different.
{% endhint %}

### Google Chrome

1. Open Chrome.
2. Go to the website that is having trouble.
3. Click the **sliders** icon to the left of the web address.
   *

       <div align="left"><figure><img src="../../.gitbook/assets/image (65).png" alt="" width="188"><figcaption></figcaption></figure></div>
4. Find  **Cookies and site data**.
5. Change the setting to **Allow**.
   *

       <div align="left"><figure><img src="../../.gitbook/assets/image (66).png" alt="" width="188"><figcaption></figcaption></figure></div>
6. Refresh the page and try again.

### If the site still does not work

* Clear cookies for that site and sign in again.
* Try a private browser window.
* Check whether a privacy extension is blocking cookies.
