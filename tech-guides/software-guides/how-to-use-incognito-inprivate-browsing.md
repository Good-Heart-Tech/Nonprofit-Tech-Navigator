# How to Use Incognito / InPrivate Browsing

## **When to Use Incognito / InPrivate Browsing**

Use a private browsing window when you need to:

* **Bypass cached data -** Helpful when troubleshooting website issues or testing changes.
* **Sign in with a different account -** Lets you log in without interfering with your main browser session.
* **Avoid storing history, cookies, or temporary data -** Ideal for sensitive browsing or testing authentication flows.
* **Verify behavior for first-time users -** Good for checking pages, redirects, or cookie banners.

## **How to Open a Private Window**

### **Google Chrome — Incognito Mode**

1. Open Chrome.
2. Use the hotkey:
   * Windows: `Ctrl + Shift + N` &#x20;
   * Mac:  `Cmd + Shift + N`&#x20;
   * Alternatively, open it manually:  Menu (⋯) → **New Incognito window**.

### **Mozilla Firefox — Private Window**

1. Open Firefox.
2. Press **Ctrl+Shift+P** (Windows) or **⌘+Shift+P** (Mac).\
   &#xNAN;_&#x4F;r:_ Menu (≡) → **New Private Window**.

***

### **Safari (Mac) — Private Window**

1. Open Safari.
2. Press **⌘+Shift+N**.\
   &#xNAN;_&#x4F;r:_ File → **New Private Window**.
