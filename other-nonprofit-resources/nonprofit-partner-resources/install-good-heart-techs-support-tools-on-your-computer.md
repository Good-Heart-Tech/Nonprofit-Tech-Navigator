# Install Good Heart Tech's Support Tools on Your Computer

To provide remote support and keep your systems running smoothly, Good Heart Tech uses a small background tool. You can learn more about why we use it here: [What’s an RMM Agent?](https://level.io/blog/rmm-agent)

Follow the steps below to install it on your computer.

***

### Windows Installation

1. **Open PowerShell as Administrator**
   * Click the Start Menu, type **PowerShell**, **right-click it**, and choose **Run as administrator**. If you do not "run as administrator", the installation will not work as expected.&#x20;
   * If prompted, click **Yes** to allow changes.
2. **Paste the Command**
   * Copy the installation command provided by Good Heart Tech. If you don't have it, email [support@goodhearttech.org](mailto:support@goodhearttech.org).&#x20;
   * To paste, **right-click** in the PowerShell window, then press **Enter**.
   * You should see a message about Level being successfully installed:
     *

         <figure><img src="../../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>
3. **Restart Your Computer**
   * After the install finishes, reboot your computer. That’s it.

***

### Mac Installation

1. **Open Terminal as Administrator**
   * Press **Command + Space**, type **Terminal**, and press **Enter**.
   * You’ll need admin permissions.
2. **Paste the Command**
   * Copy the installation command provided by Good Heart Tech. If you don't have it, email [support@goodhearttech.org](mailto:support@goodhearttech.org).&#x20;
   * Paste it into Terminal, press **Enter**, and wait for the install to finish.
3. **Accepting permissions**
   * After the installation is completed, a window will pop up, prompting you to allow Level to run. **Open System Settings** as directed.
   *

       <figure><img src="../../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>
   * Once in _System Preferences_, go to _Accessibility_, _Full Disk Access_, and _Screen Recording_, and enable Level to run. You will need to enter admin credentials.
4. **Restart Your Mac**
   * Once the process completes, reboot your Mac.
